COVID 19 Data Exploration

In the project 'COVID 19 Data Exploration', I utilized my skills in SQL and data analysis to explore global COVID-19 data using SQL Server Management Studio. I obtained up-to-date data from ourworldindata.org and used various techniques, such as CTEs and TEMP tables, as well as aliases, to manipulate and explore the data.

Through my analysis, I was able to uncover key insights about the COVID-19 pandemic, including the likelihood of contracting COVID in different countries, the number of people infected and vaccinated, and the countries and continents with the highest death rates.

This project demonstrates my ability to use SQL and data analysis to explore and understand complex data sets, as well as my proficiency in using SQL Server Management Studio and various techniques to manipulate data. It also showcases my attention to detail and my ability to uncover valuable insights from data.

As a fresher, I am excited to continue learning and growing as a data professional and to apply my skills and knowledge to real-world problems. I am confident in my ability to deliver value to any team or project that requires data analysis and exploration.
